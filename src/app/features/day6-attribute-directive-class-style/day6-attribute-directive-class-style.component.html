Class binding

- Cú pháp 1: [class.className]="variable" - Cú pháp 2: [class]="class_expression"
class_expression có thể là một string, array hoặc một object
class_expression là một string : [ngClass]="my_class_1 my_class_2 ..." chúng được viết cách nhau bởi một dấu space
class_expression là một array: [ngClass]="['my_class_1', 'my_class_2', ...]" class_expression là một object
