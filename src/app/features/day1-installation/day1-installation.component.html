Day 1: PREPARE THE WORKING ENVIRONMENT

CHUẨN BỊ

- IDE/Editor: Visual studio code

- Extension: Angular Language Service, EditorConfig for VSCode, ESLint/TSLint, NxConsole

TIẾN HÀNH

- Cài đặt Nodejs

- Sau khi cài đặt Nodejs xong thì kiểm tra Nodejs và Npm bằng câu lệnh : node -v và npm -v

- Cài đặt Angular CLI bằng câu lệnh : npm install -g @angular/cli@latest

- Sau khi cài đặt Angular CLI xong thì kiểm tra bằng câu lệnh : ng --version

KHỞI TẠO DỰ ÁN

- Sau khi cài đặt xong môi trường thì chúng ta có thể bắt đầu khởi tạo dự án bằng câu lệnh : ng new my-project

- Chạy project bằng câu lệnh : ng serve
