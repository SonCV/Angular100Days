Day 4: ANGULAR STRUCTURE DIRECTIVE - NGIF

CẤU TRÚC IF-ELSE

- Để hiển thị một phần view (template) theo điều kiện chúng ta sử dụng directive *ngIf

- Example:
Nếu bạn trên 13 tuổi thì bạn có thể xem nội dung bên dưới {{user.name}} - {{ user.age }}
