Day 3: ANGULAR DATA BINDING

INTERPOLATION

- Interpolation là cách binding data từ code ts sang view bằng cách sử dụng hai cặp dấu ngoặc nhọn lồng nhau, bên trong là một expression

- Example : binding data một object user

name: {{ user.name }}
age: {{ user.age }}
height: {{ user.height }}
